
# MyWallet

Organize your finaces in an easy and pratical way! In this app, you can learn how you spend your money and control your balance.

Try it <a href="https://my-wallet-front-topaz.vercel.app" rel="nofollow" target="_blank">here</a>.

## About

This is a web application (for mobile first) with which many people can control their balance and see where your expenses are. Below are the implemented features:

- Sign Up
- Login
- List all your finances
- Add a deposit or a payment
- Modify a deposit or a payment
- Remove a deposit or a payment
- See your balance

This project was made to test my knowledge in react routers, styled components, authentication and forms.

## How to run

1. Clone this repository
2. Clone the back-end repository <a href='https://github.com/luis-garbelotti/my-wallet-back'>here</a>
3. Follow the instructions to run the back-end <a href='https://github.com/luis-garbelotti/my-wallet-back'>here</a>
4. Install fornt-end dependencies
```bash
npm i
```
5. Run the front-end with
```bash
npm run start
```
6. You can optionally build the project running
```bash
npm run build
```
